# Movie-Recomodation
Recomodate movie using algorithm

🎬 Advanced Movie Recommendation Dashboard

An interactive, futuristic movie recommendation system built with Python, Streamlit, and TF-IDF. This project allows users to explore movies, get content-based recommendations, and visualize insights from movie descriptions with beautiful, interactive graphs.

Features

🎥 Content-Based Movie Recommendations
Recommends similar movies based on plot/overview using TF-IDF and cosine similarity.

📊 Interactive Dashboard

Histogram of movie description lengths

Top 20 movies with longest descriptions (Altair bar chart)

Optional word cloud of movie descriptions

⚡ Futuristic UI
Built with Streamlit, fully responsive and interactive.

🔍 Dynamic Search
Select any movie from a dropdown to instantly see recommendations.

Technologies Used

Python 🐍

Pandas for data manipulation

Scikit-learn for TF-IDF and similarity calculations

Streamlit for interactive dashboard

Plotly & Altair for visualization

WordCloud & Matplotlib (optional)
